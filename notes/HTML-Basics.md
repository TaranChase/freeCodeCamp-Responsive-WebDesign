# HTML FOUNDTION

- *HTML* stands for **Hypertext Markup Language**
- It is a language used for creating web pages
  

## Elements

- HTML represents content and structure of a webpage through the use of **elements**

- Most elements have an *opening* and *closing* tag. These tags are referred to as *start* and *end* tags.
  - In between the tags we have the *content.*
  - Opening and Closing tags start with a left angle bracket (`<`) and end with a right angle bracket (`>`) with a tae name placed between the angle brackets.

    `<p> Hello World! </p>`

  - Some HTML elements do not have a closing tag. These are known as **void** elements.
  
    `<img>` or `<img />`

    - The `/` in void elements does not mark the start tag as self-closing but instead is unnecessary and has no effect.


## Headings

`h1` `h2` `h3` `h4` `h5` `h6` 
These represent **headings** in HTML. They wary in font-size from *large* to *small*.

## Attributes

- These are special values used to adjust the behavior for an HTML element. 
- They provide additional information about the element.

    `<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/running-cats.jpg">`

 - `src` attribute is used to specify the location of the image.
 - It is good practice to add the `alt` attribute which is used to provide short, descriptive text for the images when an image fails to appear.

     `<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/running-cats.jpg" alt="Two tabby kittens sleeping together on a couch." />`
